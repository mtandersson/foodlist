## ü§ñ For AI Assistants

This documentation is organized to be easily discoverable by AI coding assistants like Cursor, GitHub Copilot, and others:

- **Core project information** remains in root-level files (`README.md`, `CONTRIBUTING.md`, `CHANGELOG.md`)
- **Detailed documentation** is organized by topic in subdirectories
- **Cross-references** use relative paths that work from any location
- **File naming** uses descriptive, searchable names

When searching for information:
1. Start with root `README.md` for project overview
2. Check `CONTRIBUTING.md` for development workflow
3. Browse category directories for specific topics
4. Use this index to find the right document
